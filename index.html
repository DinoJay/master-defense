Background

Current Visualisation Techniques for tags.

  • visual pleasing representation engages in search activity.

Venn Diagram

This technique depicts sets by circles and intersections by overlaps.

Thesis Aims

A new Tag Visualisation technique for Search needs to be designed with the following properties:

  • better Scaling
  • navigation structure
  • visual appealing
